# 💡 마크다운 디자인 꿀팁: 문서 구조화 전략

1. **상대 경로 사용:** 절대 경로(`C:/Users/...`)는 쓰지 마세요. 항상 프로젝트 루트를 기준으로 한 **상대 경로(`./`)**를 사용해야 깃허브에서도 링크가 깨지지 않습니다.
2. **README.md = 중앙 관제탑:** 모든 `docs/*.md` 파일은 `README.md`에서 시작하고 다시 `README.md`로 돌아올 수 있게 링크를 배치하세요.
3. **가독성을 위한 앵커 활용:** 문서가 길어질 때는 문서 상단에 `[1. 개요](#1-개요)` 처럼 섹션 링크를 모아두면, 긴 글도 웹 페이지처럼 탐색할 수 있습니다.
